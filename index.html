Sobre Este Projeto

Projeto de portfólio com foco total em aprendizado e prática de front-end, desenvolvido para consolidar conceitos de organização de código, arquitetura e consumo de dados externos em aplicações web modernas.

O projeto utiliza Vite como ferramenta de build e dev server, JavaScript Vanilla, HTML, CSS e Bootstrap para a interface. A organização dos estilos segue a metodologia BEM, garantindo padronização, legibilidade e escalabilidade do CSS. A estrutura do código é inspirada em princípios de DDD (Domain-Driven Design), com separação clara entre domínio, aplicação, infraestrutura e UI.

As imagens são servidas via Cloudinary, simulando um cenário real de uso de CDN e recursos externos, além de boas práticas de modularização e responsabilidade única. Para melhorar a experiência de seleção, o projeto utiliza um select customizado com Select2, permitindo busca rápida e interação mais eficiente com a lista de personagens.

Como funcionalidade principal, o usuário pode selecionar um personagem em uma lista interativa (com nome e ícone), e a aplicação carrega dinamicamente um carrossel com todas as skins correspondentes ao personagem escolhido. A troca de personagem atualiza o carrossel automaticamente, mantendo a experiência simples, rápida e focada no essencial.
